評價商品：
在您簽收訂單之後，請選擇確認訂單後進行評價，可以將您對我們產品和服務各方面進行評價，同時您可以可登錄TVpad官方論壇（http://www.tvpadfans.com）進行發帖分享您的購物心得，還會有驚喜哦！

修改訂單：
   訂單提交后，訂單詳情（訂購數量、收貨人、收貨地址等）均不能修改，如重要信息需要修改（如收貨地址，）請聯絡客服人員進行修改，或重新下單。

訂單查詢：
1、華揚國際網站需註冊后才能訂購，註冊成為用戶后可登錄“用戶中心”查詢您帳戶信息及訂單詳情。                         

2、您還可以將您的訂單號碼及收貨資訊告知我們的線上客服，客服人員會為您查詢您的訂單狀態。

備注：未支付訂單有效期爲1個月。每個月最後一天會對清除上個月未支付訂單（比如：2月29號清除1月1-31號的未付款訂單）；1月份未付款客戶若需再訂購，重新下單。爲了您盡快享用到TVpad帶來的全新視覺體驗；請及時支付。
