  • 重構電視未來，TVpad2為變革而生
  • 出處：TVpad 作者： 發表時間：2013-01-10 17:40:30

  •    啟創科技領銜電視未來，近日正式發布了劃時代突破性產品--全"芯"系列TVpad2智能機頂盒，并重磅推出家庭卡拉OK、網絡通話、體感游戲等全新互動功能，相對前系列產品在硬固件及用戶體驗方面變革不少。

       外觀上，繼續沿用M121S"便捷小巧，綠色環保"的理念；技術上，引入了新一代無線技術，內置wifi從此解決M121S無線網絡連接問題；硬、固件上，配置高性能芯片，搭載深度化智能操作系統，運行速度相比M121S提升了近60%，體驗更流暢；應用上，提供全開放性的TVpad Store應用平臺，提供海量第三方開發商提供的正版優質應用，確保您盡情暢享新聞資訊、影視娛樂、體育快報、少兒動漫、動感游戲、卡拉ok、網絡通話等豐富體驗。同時，相對于M121S，TVpad2更著重于互動體驗功能的完善；體感游戲功能為您帶來運動快樂，卡拉OK功能滿足您淋漓盡致的互動享受，網絡通話功能讓您全球免費暢聊……

            

    海量應用，高清穩定片源

       TVpad2應用商城升級為了TVpad Store，相比前系列產品，TVpad Store擁有海量第三方開發商提供的正版優質應用，每一款App都通過了啟創科技與運營商的認證審核，擁有TVpad2，用戶可以盡情下載自己喜歡的應用，和家人一起暢享新聞資訊、影視娛樂、體育快報、少兒動漫、體感游戲、家庭KTV、網絡通話等豐富體驗，無需再為應用功能是否穩定而擔憂。

    全"芯"升級，操控流暢超乎想象

       在硬、固件方面，TVpad2實現了全面升級，徹底解決了前系列產品反應慢、視頻卡屏、黑屏過度的問題，性能上比M121S提升了近60%；從開機、切換應用，到觀看直播節目和玩轉體感游戲，讓用戶進行的每一項操作都感覺流暢自然，流暢體驗超乎想象。

    體感游戲，淋漓盡致運動體驗

       在互動體驗方面，TVpad2配置了高性能圖形處理器，特別開設專屬游戲應用，用戶從此在電視上也能體驗體感手柄游戲，在家就能享受淋漓盡致的運動體驗。

    內置Wi-Fi，輕松便捷更穩定

       全新TVpad2引入了新一代無線技術，內置高速Wi-Fi無線網卡和高增益的Wi-Fi天線，解決了前系列產品無線網絡連接問題，用戶從此可以告別外置無線設備，告別繁雜的網線，在無線網絡覆蓋范圍內，隨時享受TVpad2帶來更輕松便捷的體驗服務。

    家庭KTV，唱享娛樂生活

       TVpad2全新推出家庭KTV功能，愛音樂的用戶只需將TVpad2同時連接電視與家庭影音設備，就可以隨時將客廳轉變為豪華KTV，在家就能暢享KTV的愉悅氛圍，不用再為花錢、找地點而困擾。

    網絡通話，全球免費暢聊（即將推出）

       TVpad2即將為用戶提供網絡通話功能，用戶可以免費撥打，暢聊全球，隨時與家人朋友通通電話、煲煲電話粥，再也不用為長途電話的高昂費用而擔心。
