如何使用禮品券：
禮品券獲取渠道：（原優惠券可以做禮品券使用，買即送超值礼品大禮包，不可優惠金額）

A：TVpad產品包裝箱內可獲得

B：老用戶通過電子郵箱可獲得

C：合作媒體可獲得

如何使用禮品券：


1.通過線上訂購TVpad產品時，在填訂購信息時請選擇“使用禮品券”並輸入禮品券碼即可，系統審核後會自動對應贈送禮包；



2.通過電話訂購TVpad產品時，請將您所擁有的禮品券內容告之我們的客服人員，並將禮品券驗證碼準確報出，客服人員將協助您下訂單并給與相應禮包內容。

安全性：

1.您的整個購物過程，我們會保證安全第一，將會有華揚國際授權的員工處理您的訂單，並且保證您的資訊處於高度安全的狀態。    
2.華揚國際尊重客戶的隱私權，在任何情況下，我們都不會將您的資訊出售或洩露給任何第三方。我們通過華揚國際網站及其他途徑所得到的所有的客戶資訊僅被用來處理您的訂單。

3. 「曲奇」我們使用「曲奇」（即cookies，當你進入某個網站時，網站伺服器傳送到你電腦瀏覽器， 並儲存在電腦硬件中的少量資料）以令你在華揚商城購物更方便。當你再次進入（譽坤）華揚國際， 我們的「曲奇」能辨認你的身份，令我們的服務更個人化。我們將會根據你的個人資料，提供更適合你需要的產品和服務。大部份的瀏覽器皆自動接受「曲奇」。你也可以在你電腦的"preference"或"option"「選項」 上選擇不接受「曲奇」。如果你選擇不接受「曲奇」，你將無法在（譽坤）華揚國際購物，但你​​仍可以瀏覽我們網站的部份網頁。
 

購物流程：
線上訂購：

》瀏覽商品
》決定購買，點擊“立即購買”按鈕或“加入購物車”繼續選購商品
》核對購物車后跳轉至”去結算“
》填寫收貨信息及選擇付款方式及是否使用優惠券
》提交訂單，完成支付
》確認收貨
》至用戶中心給予交易評價
》完成訂購

备注：一张订单只对一个发货地址。
