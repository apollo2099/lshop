    若产品运送到中国以外的国家，您可能需要缴纳关税。我们会在商业发票上标明申报价值，为尽量减少关税，此价值并非是您的支付价格。任何由报关产生的费用由您来承担。 同时，涉及到个别国家清关情况的不同，如您需要降低申报价值，请及时与客服联系【售后服务专线：0755-26520101（国内）；00852-31-158-518（海外）】或备注说明。
