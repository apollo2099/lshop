公司简介：
公司介绍与品牌实力　　
    banana TV是深圳市金香蕉信息技术有限公司旗下品牌，公司总部位于深圳，在深圳、上海、北京三地拥有数百人的研发中心，拥有上百项国际、国内专利技术。5年多的技术研发和市场实践，于2014年初正式推出最新的基于Andriod系统深度定制的智能机顶盒。 
    公司以“服务与创新”为核心价值观，以“家文化的倡导者”为品牌理念，深度挖掘家庭视听需求，为家庭提供丰富多彩的电视应用，满足家庭成员对教育、娱乐、育儿、游戏等个性化的需求，营造共享、共融、和谐的家庭文化氛围，畅享幸福多姿的家庭文化生活。

家文化的倡导者　　　
    家是社会的基础，家文化更是中华文化的根本，是中华民族千百年来传统的延续。智能化时代的今天，智能终端设备的风靡，改变了人与人之间的关系，融洽的家庭氛围正因个人电子产品过渡使用而淡化，家庭文化被逐渐侵蚀。 
    banana TV从家出发，重视对中国传统家庭文化的承袭与创新，倡导快乐、温馨、分享、传承的和谐家文化。始终关注家庭幸福，孩子成长，家人健康，塑造良好的家庭氛围，致力于成为每个家庭成员情感互联的纽带，让每个家庭、每个成员都能从中寻找到共享、共融的空间。
